How to Use

  1. Enter your Anthropic API key above
  2. Go to a LinkedIn job posting
  3. Click the "Tailor Resume" button that appears on the page, or click the extension icon
  4. The job details will be auto-extracted
  5. Click "Generate Tailored Resume" to create a customized version
  6. Copy the result or download it as a text file
